# Pengeluaran Que
Project latihan untuk belajar mengenai Room Database dan mengenalkan Android Architecture Components.

## Android Architecture Components
* Lifecycle
* LiveData
* Room
* ViewModel

## Fitur
* Create
* Read
* Update
* Delete
